    Activity

    The Activity document is used to enter a specific activity and mark the result of it. To create “Activity” go to Customers >> Contacts & Tasks >> Create New Activity

    ![Activity](pictures/Create New Activity.png) Fill in the following fields:

    • System Type - you can set whether to be Communication, Meeting or Task.

    ![Activity](pictures/System Type.png)

    • Subject - you can enter a short text that will be the topic of the activity.

    • Start Time - you can set the date and time for starting the activity.

    ![Activity](pictures/Start Time.png)

    • End Time - you can enter the date and time for ending the activity, the Deadline Time field will be calculated automatically

    ![Activity](pictures/End Time.png)

    • Deadline Time - you can enter the planned duration in minutes, then the End Time field will be calculated automatically.

    ![Activity](pictures/Deadline Time.png)

    After filling in the data Release the document.

    ![Activity](pictures/Release the document.png) ##To view all created Activities: Go to Customers >> Contacts & Tasks >> Activities Navigator Press the Display button. All created activities will be loaded.

    ![Activity](pictures/Activity Navigator.png) By double-clicking on the selected activity, you can open it to view it.
