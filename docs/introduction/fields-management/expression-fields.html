    Expression fields

    Expression fields are created by the customer in the database. You can use them to generate formulas and create new fields where the formula calculations are applied. The formula cannot include values from other expression fields.

    Open the expression fields form from: Data→Expression fields

    Example:

    • Field name: VAT
    • Title: VAT
    • Formula: press the button

    A formula editor opens. It is separated into three regions – the first one shows the formula; the middle one – various actions than can be executed; the last one is a fields panel where the fields included in the formula can be selected.

    Example:

    • select Amount to pay
    • select the ‘-’ symbol
    • select Lines total

    The value in the new field will be summed by the formula: Amount to pay – Lines total = VAT field

    Click OK to close the Expression editor or the Expression fields form.

    The VAT field shows up in the data panel.
