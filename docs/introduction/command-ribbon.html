    Command Ribbon

    The Command Ribbon is a modern alternative to the classic menus. The actions in it are visualized through buttons. It is easy to use with a single tap or mouse click. What is special about the Command Ribbon in ERP.net Windows Desktop Client is its adaptive context. The buttons change and provide specific functions, needed for the workflow you are currently working with. This way, you always see the exact buttons you need.

    The Command Ribbon contains two main pages - Home and File, which you can find in any system Form. You can also add personalized buttons for quick access. This is an additional row, which contains fixed and always visible buttons. It is created by the user and therefore it is specific for every user and every company. You can add buttons from different tabs of the Command Ribbon there.
